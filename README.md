
# KhabarBolo

A local food delivery web app 

## Test deployment 

https://test-khabarbolo.netlify.app

## Run Locally

Clone the project

```bash
  git clone https://github.com/unsuredev/khabarBolo.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
